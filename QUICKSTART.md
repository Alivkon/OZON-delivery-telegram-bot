# 🚀 OZON Status Bot - Быстрый старт

## 📥 Скачивание проекта
```bash
git clone <repository-url>
cd OZON
```

## ⚡ Быстрая установка

### 1. Проверьте систему
```bash
python check_system.py
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте бота
```bash
cp .env.example .env
# Отредактируйте .env файл своими данными
```

### 4. Запустите бота
```bash
python ozon_bot.py
```

## 🔧 Что нужно для работы:
- ✅ Python 3.8+
- ✅ Telegram Bot Token (получить у @BotFather)
- ✅ OZON API ключи (Client-Id и Api-Key)

## 📱 Использование:
1. Напишите боту `/start`
2. Введите номер заказа
3. Нажмите "✅ Доставлено"
4. Получите результат!
